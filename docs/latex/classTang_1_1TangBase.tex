\doxysubsection{Detailed Description}
The base class for the Tang programming language. 

This class is the fundamental starting point to compile and execute a Tang program. It may be considered in three parts\+:


\begin{DoxyEnumerate}
\item It acts as an extendable interface through which additional \char`\"{}library\char`\"{} functions can be added to the language.

It is intentionally designed that each instance of \mbox{\hyperlink{classTang_1_1TangBase}{Tang\+Base}} will have its own library functions.
\item It provides methods to compile scripts and templates, resulting in a \mbox{\hyperlink{classTang_1_1Program}{Program}} object.
\item The \mbox{\hyperlink{classTang_1_1Program}{Program}} object may then be executed, providing instance-\/specific context information ({\itshape i.\+e.}, state). 
\end{DoxyEnumerate}

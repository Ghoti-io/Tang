\doxysubsection{Detailed Description}
A container that acts as a resource-\/counting garbage collector for the specified type. 

Uses the \mbox{\hyperlink{classTang_1_1SingletonObjectPool}{Singleton\+Object\+Pool}} to created and recycle object memory. The container is not thread-\/safe.

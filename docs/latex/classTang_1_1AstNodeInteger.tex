\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classTang_1_1AstNode}{Ast\+Node}} that represents an integer literal. 

Integers are represented by the {\ttfamily Tang\+::integer\+\_\+t} type, and so are limited in range by that of the underlying type.

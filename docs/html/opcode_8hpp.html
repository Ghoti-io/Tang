PC #: set pc to PC #.

PC #: read val, if false, set pc to PC #.

JMPF_POP 

PC #: pop val, if false, set pc to PC #.

PC #: read val, if true, set pc to PC #.

JMPT_POP 

PC #: pop val, if true, set pc to PC #.
